LUXÉ Fashion Template

Thank you for purchasing this template.

Installation

  1. Download the template ZIP file
  2. Extract the files
  3. Open index.html in your browser

Project Structure

luxe-fashion-template
│
├── index.html
├── css
│   └── style.css
├── js
│   └── script.js
├── images
└── docs
    └── documentation.html

How to Edit Products

Open the file: js/script.js Then edit the product list. Example: 
{
name: "Luxury Jacket",
price: 120,
image: "images/jacket.jpg"
}

Customize Design

To change colors or layout edit: css/style.css Example: 
body{
background:#ffffff;
}
